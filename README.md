# Employee-Payroll-cpp-1
## 📖 Description

A console-based application built in C++ to manage employee records and salary details. It supports adding, updating, deleting, and viewing employee and salary data.

## ⚙️ Features

Add / Update / Delete Employees
Search Employee by ID
Manage Salary (Full-time & Part-time)
Calculate Net Salary

## 🛠️ Tech Used

C++
Standard Template Library (vector, string)

## ▶️ How to Run

g++ employee_payroll.cpp -o app
./app

## 📂 Structure

employee_payroll.cpp → Complete program

## ⚠️ Limitations

No file/database storage (data lost after exit)
Basic UI (console only)
